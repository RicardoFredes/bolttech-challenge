# Bolttech challenge

This repository is a technical challenge from Bolttech.

## Dependencies

- Node JS >= 14.18.0

## Running

### Development

Run the command below to run the project in development environment:

```bash
$ yarn install
```

Run the commands below in separate terminals to run the backend and frontend:

Backend:

```bash
$ yarn back:dev
```

Frontend:

```bash
$ yarn front:dev
```
